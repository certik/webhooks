How to add a repository and users

To add a new repository, set up a post-receive hook on your repository (use the url of this page) as explained in the Post-Receive Guide at GitHub. That's the best way, because we will be kept updated about all changes to the repository. Once we receive a hook, the corresponding repository and users will be automatically created.

Another option is to create the repository manually on the repositories list page.

About

This is a web application that provides an alternative interface to the repositories hosted at GitHub.

It uses the GitHub web API to access information about the repositories and then caches it in the database (everything runs on the Google App Engine), so it's very fast.

This application is opensource (BSD licensed), source code is at http://github.com/certik/webhooks
