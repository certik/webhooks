About

This is a web application that provides an alternative interface to the repositories hosted at GitHub.

It uses the GitHub web API to access information about the repositories and then caches it in the database (everything runs on the Google App Engine), so it's very fast.

This application is opensource (BSD licensed), source code is at http://github.com/certik/webhooks

Add a repository

To add a new repository, set up a post-receive hook on your repository as explained in the Post-Receive Guide at GitHub. This is important, so that we are kept updated about all changes to the repository. {% endblock %}
